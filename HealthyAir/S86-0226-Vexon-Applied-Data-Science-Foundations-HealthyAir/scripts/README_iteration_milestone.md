# Iteration Milestone — For & While Loops

This folder contains a minimal Python script demonstrating `for` and `while` loops, control flow (`break`, `continue`), and a safe example illustrating how to avoid infinite loops.

Files
- `scripts/iteration_milestone.py`: runnable examples that print outputs to observe behavior.

Run
```bash
python scripts/iteration_milestone.py
```

What to include in your ~2-minute video walkthrough
- Run the script while screen-recording so viewers see the console output.
- Show a short for-loop example (the script includes `for_range_example` and `for_list_example`).
- Show a short while-loop example (`while_countdown`).
- Demonstrate use of `break` or `continue` (`while_with_break_continue`).
- Explain how the `infinite_loop_safe_demo` uses a safety counter to prevent a real infinite loop.

Video checklist
- ~2 minutes long.
- Screen-facing and clearly visible (console and code).
- Narrate what each loop does and why it terminates.

Notes and safety
- Do not run real infinite loops while recording — use the safety demo provided.
- Keep examples simple and readable; avoid deep nesting.

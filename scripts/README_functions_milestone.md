# Functions Milestone — Defining & Calling Functions

This folder contains a minimal Python script demonstrating how to define and call functions in Python. The examples cover:
- simple `def` functions
- return values
- default parameters
- `*args` and `**kwargs`
- local vs global scope (and the `global` keyword)

Files
- `scripts/functions_milestone.py`: runnable examples that print outputs to observe behavior.

Run
```bash
python scripts/functions_milestone.py
```

Video walkthrough (~2 minutes) checklist
- Show defining a function and explain the `def` signature.\n
- Call the function with arguments and show returned values.\n
- Demonstrate default parameters and `*args`/`**kwargs`.\n
- Explain local vs global variables and show `modify_global()` demonstration.\n
Notes
- Keep explanations short and focused; avoid modifying globals in production code unless necessary.
